<h3 align="center">Unified calendar app for The Lesli Framework.</h3>

<hr/>

Version 0.1.0


#### Quick start

```shell
# Add LesliDriver engine
bundle add lesli_driver
```

```shell
# Setup database
rake lesli:db:setup
```

```ruby
# Load LesliDriver
Rails.application.routes.draw do
    mount LesliDriver::Engine => "/driver"
end
```
